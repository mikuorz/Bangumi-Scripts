# Bangumi-Scripts
A collection of userscripts to enhance the Bangumi website experience.

# [Mikuorz 的 Bangumi 主页](https://bgm.tv/user/pereza)

## [Bangumi 讨论区图片尺寸控制](Bangumi%20%E8%AE%A8%E8%AE%BA%E5%8C%BA%E5%9B%BE%E7%89%87%E5%B0%BA%E5%AF%B8%E6%8E%A7%E5%88%B6.user.js?raw=true)

**功能介绍：**
本脚本用于优化 Bangumi 讨论区的阅读体验，自动限制超大图片的显示宽度，防止撑破页面布局。支持点击图片放大查看原图。

**用法：**
安装后在讨论区页面自动生效。
